工作经历

广州市为人软件科技有限公司 （ 2018年4月 ~ 至今）

招商数据采集公众号项目

招商数据采集项目是一个数据可视化的微信公众号项目。
我在此项目独立负责了企业信息图谱的数据可视化的开发部分，由于该图谱的个性化需求较高，所以没有采用已有的一些可视化框架进行开发，而是使用D3.js库为辅助与vue相配合进行开发，最后再将其封装为一个vue的子组件，方便调用。由于项目中一些复用的DOM比较多，因此项目搭建方面采用了Vue CLI+Weui+Less进行构建,使用Webpack、Node.js辅助打包管理项目，使用D3.js库生成企业自画像图谱。

窝米行业采购招标信息公众号项目

这是一个面向付费用户的微信端H5的行业采招信息查询的公众号。
我在此项目独立负责了所有前端部分的开发。由于页面设计是以的是公司以前已有项目为基础进行二次设计并且当时项目比较紧急，所以我就将原有项目其中的一些相似的页面结构进行修改、重构。将页面改为使用Vue为框架，引入Require.js辅助加载所需的脚本，再配合Weui搭建页面，使用axios获取后台数据、localStorage缓存用户检索历史、使用微信开放接口实现用户支付。

其他项目

采用Vue+Element+Echarts框架独立开发PC版窝米后台管理系统
负责窝比餐厅小程序的维护与迭代

汕头市深蓝软件有限公司 （ 2017年2月 ~ 2018年3月 ）

我的微幸福公众号项目

这是一个用于记录用户每日幸福事件的微信公众号，用户可在该公众号记录事件、查看他人事件、点赞、评论，组圈子。
我在此项目独立负责了初级版本前端部分的开发工作。
项目使用Jquery+Weui为基础框架实现页面布局和交互，用rem/百分比/flex布局实现页面自适应，使用jQuery File Upload实现文件/视频上传，使用LocalStorage实现用户编辑存储，采用ajax获取后台数据

吃多多微信小程序项目

吃多多是一个用户采用“集卡”、“集字”/ 购买会员卡等方式获取优惠的一个生活服务类的小程序。
我在此项目独立负责了前端方面的开发工作。
项目使用Weui为辅助开发页面，用百分比/flex布局实现页面自适应，使用template/import引入方式归纳页面可复用组件。这是我的第一个微信小程序项目，这个项目的开发周期比较短，因此我是边学习边开发这个项目，通过这个项目让我掌握了基本的小程序开发技术。
